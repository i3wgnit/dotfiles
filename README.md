# dotfiles

Clone this repo, install submodules, run `install` to set up.

This dotfiles uses [Dotbot](https://github.com/anishathalye/dotbot/) for installation.

## Dependencies

- st
- rofi
- i3
  - i3 gaps
  - i3blocks gaps
- fonts
  - misc ohsnap
  - iosevka
    - iosevka term
    - iosevka nerd font

### Optional

- ranger
- calcurse
- calculator
  - R
  - qalc
- browser
  - qutebrowser
  - firefox
- mpv
- mutt
