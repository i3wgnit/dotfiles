# dotfiles

Clone this repo, install submodules, run `install` to set up.

This dotfiles uses [Dotbot](https://github.com/anishathalye/dotbot/) for installation.

## Dependencies

- st
- dwm
  - i3blocks gaps
- dmenu
- fonts
  - misc ohsnap
  - iosevka
    - iosevka term
    - iosevka nerd font

### Optional

- calcurse
- compton
- browser
  - qutebrowser
  - firefox
- calculator
  - R
  - qalc
- mpv
- mutt
- ranger
- redshift
